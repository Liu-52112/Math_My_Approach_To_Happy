## 介绍
作为一名计算机系的数学爱好者，喜欢刷一些数学视频，看一些数学文章，但刘某不才，一遍总记不得，事后还想再回顾，证明，推导，但也不知道原视频在哪。国内外各大视频网站收藏夹，点进去之后，一辈子都不会再看，而且各平台之间也不会互相收录，所以想要建一个自己的数学知识库。

数学带给我的感受正如它的英文Math一样，是My_Approach_To_Happy（Math的缩写）。

也许，我天赋平平，但从喜欢上数学的那一刻开始，我有了对抗无聊的工具。

数学给人精确，给人欣喜，给人振奋，所以准备好了么？

Let‘s begin
